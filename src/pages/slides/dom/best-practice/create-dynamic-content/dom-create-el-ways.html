Добавление динамического контента

Итого сейчас мы знаем 3 способа создания и добавления контента динамически:

  1. createElement + Element API

    В таком варианте вся структура создается посредством document.createElement, с последующим выставлением всех параметров и атрибутов при помощи Element API

  2. innerHTML + Template String

    При этом явно создается лишь верхний контейнер. Внутренняя разметка вставляется как innerHTML используя TemplateString. При этом нам доступна интерполяция, и разметка динамического фрагмента видна явно.

  3. Template HTML

    Здесь мы полностью сохраняем разметку обособленной. Но при этом нам приходится работать с относительно более сложным API.
