Browser Object Model

Объектная модель браузера (анг. Browser Object Model (BOM)) — это часть JavaScript, которая позволяет скрипту взаимодействовать с браузером. BOM представляет объекты, через свойства и методы которых можно управлять внешним видом и поведением веб-страницы и её элементов.
