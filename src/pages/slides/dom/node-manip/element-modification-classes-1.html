Модификация списка CSS-классов элемента

Способов доступа к списку CSS-классов существует несколько:

  • параметр className [Element.prototype], сязанный с атрибутом "class".
  • динамическая коллекция classList [Element.prototype] (о ней поговорим далее)

Формально доступиться к списку классов можно и уже известным способом: методами setAttribute/getAttribute по имени "class".
