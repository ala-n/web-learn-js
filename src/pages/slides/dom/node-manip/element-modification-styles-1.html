Модификация стилей элемента

Для получения и установки инлайновых стилей используется свойство style [HTMLElement]. Это свойство является экземпляром прототипа CSSStyleDeclaration. CSSStyleDeclaration - не псевдо-массив, а несколько более сложная структура, это хранилище ключ-значение с динамическим подсчетом их кол-ва (доступно свойство length). В качестве ключа используется название css опции в CamelCase. Т.е.:

  • для доступа к свойству color используется такой же ключ color
  • а для доступа к свойству margin-top ключ будет marginTop

В качестве значения используется строка, соответствующая тем же правилам, что и оригинальное значение CSS-свойства.
